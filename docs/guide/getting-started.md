# Getting Started

Welcome to the **Rspress Graph View** documentation!

This plugin creates an interactive [graph visualization](./graph-view.md) of your documentation, similar to [Obsidian's graph view](https://obsidian.md).

## Features

- Interactive force-directed graph
- Click nodes to navigate
- Current page highlighting
- Dark mode support
